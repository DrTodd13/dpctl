Detailed Description

This header declares a C API to create Sycl program an interoperability program defined in OpenCL. In future, API to create interoperability kernels from other languages such as Level-0 driver API may be added here.

Create a Sycl program from an OpenCL SPIR-V binary file.

Sycl 1.2 does not expose any method to create a sycl::program from a SPIR-V IL file. To get around this limitation, we need to use the Sycl feature to create an interoperability kernel from an OpenCL kernel. This function first creates an OpenCL program and kernel from the SPIR-V binary and then using the Sycl-OpenCL interoperability feature creates a Sycl kernel from the OpenCL kernel.

The feature to create a Sycl kernel from a SPIR-V IL binary will be available in Sycl 2.0 at which point this function may become deprecated.

Parameters
CtxAn opaque pointer to a sycl::context
ILSPIR-V binary
Returns
A new SyclProgramRef pointer if the program creation succeeded, else returns NULL.
