Detailed Description

This file was copied over from the LLVM-C (include/llvm-c/DataTypes.h). This file contains definitions to figure out the size of HOST data types. This file is important because different host OS's define different macros, which makes portability tough. This file exports the following definitions:

[u]int(32|64)_t : typedefs for signed and unsigned 32/64 bit system types [U]INT(8|16|32|64)_(MIN|MAX) : Constants for the min and max values.

No library is required when using these functions.
