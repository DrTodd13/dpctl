Function DPPLQueueMgr_GetQueue¶

Function Documentation¶

DPPLSyclQueueRef DPPLQueueMgr_GetQueue(DPPLSyclBackendType BETy, DPPLSyclDeviceType DeviceTy, size_t DNum)¶

Get a sycl::queue object of the specified type and device id.

Return

A copy of the sycl::queue corresponding to the device is returned wrapped inside a DPPLSyclDeviceType pointer. A runtime_error exception is raised if no such device exists.

Parameters

  • DeviceTy: The type of Sycl device (sycl_device_type)

  • DNum: Device id for the device (defaults to 0)
