Function Documentation¶

DPPLSyclQueueRef DPPLQueueMgr_GetQueueFromContextAndDevice(DPPLSyclContextRef CRef, DPPLSyclDeviceRef DRef)¶

Creates a new instance of SYCL queue from SYCL context and SYCL device.

The instance is not placed into queue manager. The user assumes ownership of the queue reference and should deallocate it using DPPLQueue_Delete.

Return

A copy of the sycl::queue created from given context and device references.

Parameters

  • CRef: Sycl context reference

  • DRef: Sycl device reference
