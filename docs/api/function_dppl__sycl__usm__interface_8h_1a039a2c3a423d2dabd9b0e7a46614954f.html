DPPLSyclUSMRef DPPLaligned_alloc_device(size_t alignment, size_t size, const DPPLSyclQueueRef QRef)¶

Create USM device memory.

Return

The pointer to USM device memory with requested alignment. On failure, returns nullptr.

Parameters

  • alignment: Allocation’s byte alignment

  • size: Number of bytes to allocate

  • QRef: Sycl queue reference to use in allocation
